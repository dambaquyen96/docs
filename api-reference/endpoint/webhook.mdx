---

<Note>
  Webhook xây dựng nên là Restful HTTP Service, nhận body request dạng application/json
</Note>

### Mẫu request body được gửi về Webhook
```json
{
  "session_id": "...",
  "verify_code": 200,
  "verify_result": "success",
  "metadata": "...",
}
```
