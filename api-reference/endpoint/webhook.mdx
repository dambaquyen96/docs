<Note>
  Webhook cần xây dựng là Restful HTTP Service, nhận body request dạng application/json
</Note>

### Mô tả body request được gửi về Webhook
| Trường | Kiểu dữ liệu | Mô tả |
| -------- | -----------|-------------------------- |
| session_id | string | Mã định danh của phiên xác thực |
| verify_code | interger | Mã kết quả xác thực OTP, là 1 trong các giá trị: <br/>- 200 : Xác thực thành công <br/>- 301: Xác thực thất bại - KH có nhấc máy <br/>- 401: Xác thực thất bại - Không kết nối được KH |
| verify_result | string | Nội dung kết quả xác thực |
| metadata | string | Thông tin dữ liệu được bổ sung tại request tương ứng (nếu có) |


```json Example
{
  "session_id": "1806643218116882432",
  "verify_code": 200,
  "verify_result": "success",
  "metadata": "{}",
}
```
